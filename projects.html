ImHere

Lead Developer, Database Manager
September 2016 - July 2017
 ImHere is an online attendance management system which provides a fast, simple, and secure method to take student attendance in a classroom setting. ImHere is written in Python2 and uses Flask as its web microframework. Persistent data is stored in a Postresql database. ImHere also uses Google's SSO for account creation/management, making it very easy to jump right in.

StockQ

Lead Developer, Language & Test Guru
October 2016 - January 2017
 StockQ is a general purpose programming language that compiles to LLVM IR. It is an imperative language that looks syntactically like a mix of C and Python. It excels in giving the user tools to quickly run mathematical calculations, which are added at a low level in the language. StockQ is best used to manage stocks and run financial operations on them.

ChampDB

Along with the search, filter, and add/remove processes that it provides, ChampDB also creates new composite types, utilizes array attributes, and employs triggers. Implemented in Flask.
